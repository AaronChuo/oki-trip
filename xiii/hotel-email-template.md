# 飯店詢問信樣板（備品リクエスト＋アクティビティ予約）

## 使用方式

每次寄信只需替換 `《》` 內的欄位，其餘不動：

| 欄位 | 說明 | 範例 |
|---|---|---|
| 《ホテル名》 | 飯店日文名（官網抬頭為準） | ホテルマリンピアザオキナワ |
| 《氏名》 | 預約者姓名（與訂單一致） | Cho Cheng Hsien |
| 《宿泊期間》 | 日式寫法＋星期＋泊數 | 2026年8月1日（土）～8月4日（火）（3泊） |
| 《予約番号》 | 訂單編號 | IY1605031451 |
| 《予約サイト》 | 訂房來源日文名 | 一休.com／楽天トラベル／じゃらん／Agoda／公式サイト |
| 《宿泊人数》 | （可選）大人・小孩人數與年齡 | 大人2名・子供1名（3歳） |

- 訂房網站中文對照：一休 → 一休.com、Rakuten → 楽天トラベル、Jalan → じゃらん。
- 若該飯店**沒有活動設施**：刪除「■ アクティビティの予約について」整段，件名改為「【備品リクエストのお問い合わせ】」。
- 備品清單可自由增減，常用品項：ベッドガード（床圍）、子供用アメニティ（兒童備品）、子供用スリッパ（兒童拖鞋）、加湿器（加濕器）、ベビーベッド（嬰兒床）、電子レンジ（微波爐）、哺乳瓶消毒器（奶瓶消毒鍋）。
